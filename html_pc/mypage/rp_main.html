릴레이팍팍 유의사항

  • ·릴레이팍팍 종료 안내 : 본 프로모션은 21년 6월 기준 6개월 달성자부터 순차 종료될 예정입니다. 기존 참여자는 완주 시까지 기존 그대로 참여 가능합니다.
  • ·릴레이팍팍 실적 대상 : 쇼핑엔티몰 주문건에 해당 (상담원, ARS, 리모컨, 카카오톡 제외)
  • ·릴레이팍팍 실적 산정 : 최종 실결제금액 기준 (할인쿠폰 제외, 적립금 포함)
    ex) 할인쿠폰 제외 : 10만원(상품금액) - 1만원(쿠폰금액) = 9만원(실결제금액) : 실적 9만원
    ex) 적립금 포함 : 10만원(상품금액) - 1만원(적립금) = 9만원(실결제금액) : 실적 10만원
  • ·취소/반품 실적 제외 : 릴레이팍팍 적립금 지급일(20일) 기준으로 취소/반품으로 인한 월 구매금액 10만원 미달시 대상 제외
  • ·적립금 지급일 : 달성월 기준, 익월 20일
  • ·적립금 유효기간 : 지급일로부터 30일까지 (기간 이후 자동 소멸)
  • ·적립금 지급/사용 대상 제외 : 무형상품(보험,렌탈,여행,렌터카,시공 등), 순금(골드바), 상품권, 휴대폰, 특가딜(쇼타임딜 등) 및 일부 상품
  • ·적립금 사용기준 : 결제금액의 50까지, 최종결제금액 2만원부터 사용가능(5월중 시행예정)
